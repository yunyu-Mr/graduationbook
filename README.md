# 序言
致各位骚年：

毕业在即，即将离去。我想尽自己所能为每个同学留下一段寄语，留下一句诚挚的感谢，或是一句衷心的祝福。  
也许毕业以后我们当中的许多人这辈子再也无法相遇。但是，在人生最羞射的四年，能和诸君一起留下印记，足以让我，永生难忘。  
以后遇到困难，别忘了还有我骚铭啊，能帮的一定帮！创业了也别忘了骚铭啊，我很乐意去搬砖的！

祝好，  
骚铭！
